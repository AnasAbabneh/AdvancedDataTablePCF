A high-performance, React-based PCF data grid designed to overcome the limitations of standard Power Apps grids.
AdvancedDataTablePCF provides a flexible, dynamic, and scalable table experience for Model-Driven and Power Apps scenarios, supporting large datasets with smooth rendering and professional UI behavior.

The control dynamically adapts to Dataverse datasets without hard-coded columns and is built to support advanced features such as multi-column sorting, virtual scrolling, column resizing, frozen columns, and row selection. It is optimized for enterprise use cases where performance, usability, and extensibility are critical.

Ideal for replacing standard subgrids and views in complex business applications where advanced data visualization and interaction are required.
